== Description ==
This is an extension for WooCommerce to accept cryptocurrencies via Pecunia Wallet.

== How to use ==
Just install from the WordPress marketplace. Or, alternatively, manually from sources:
### 1. Put sources into folder `pecunia-payment-gateway-for-woocommerce` and zip or download the archive [here](https://github.com/Pecunia-Wallet/pecunia-woocommerce/releases/latest)
### 2. Install and activate plugin
<img width="1800" height="867" alt="Screenshot 2026-06-05 122525" src="https://github.com/user-attachments/assets/36b19c7e-afe3-4dab-b5e4-9c7078ed75f2" />

### 3. Go to WooCommerce Settings > Payments, configure and enable the gateway
<img width="1775" height="859" alt="Screenshot 2026-06-05 122837" src="https://github.com/user-attachments/assets/75ad4f46-f640-4ad2-92bd-dd55ebb834fd" />

### 4. That's it!

== External services ==
The plugin uses the Pecunia Wallet API to function.

It creates and tracks invoices using provided settings values.
